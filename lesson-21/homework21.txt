Homework: choose and implement one pattern
Task:
Choose any design pattern from the lesson topics (Singleton, Factory Method, Observer, Strategy, etc.) and implement it in Go.
Requirements:
• The code must be in a separate package or file.
• Add comments explaining how the pattern works.
• Provide an example of usage in main().
Example formulation:
1. Describe the pattern (what problem it solves).
2. Implement the structure/interfaces.
3. Show an example of it in action.
Goal: reinforce understanding of design patterns and learn how to apply them in real code.
