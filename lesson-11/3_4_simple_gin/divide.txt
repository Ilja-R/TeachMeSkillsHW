Task:
Extend the Gin service with the following logic:
1. The POST /divide handler accepts JSON:
{
  "a": 10,
  "b": 2
}
2. Performs the division a / b and returns the result:
{
  "result": 5
}
3. If b == 0, return an error:
{
  "error": "division by zero"
}

2.
